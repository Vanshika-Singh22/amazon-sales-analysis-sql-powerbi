# Amazon Sales Analysis using SQL and Power BI
## Project overview
This project analyzes Amazon sales data using SQL and Power BI. The objective is to identify sales trends, top-performing products, customer purchasing patterns, and business insights that can help improve decision-making.
## Objectives
- Analyze overall sales performance.
- Identify top-selling products and categories.
- Examine order status trends.
- Analyze sales by city and state.
- Generate business recommendations.
- ## Dataset Description
The dataset contains Amazon sales transaction records with information such as:
- Order ID
- Date
- Product Category
- Size
- Quantity
- Sales Amount
- City
- State
- Order Status
- Fulfillment Method
- ## Tools Used
- SQL
- Power BI
- Excel/CSV
- GitHub
- ## SQL Analysis
1. Order Status Analysis
2. Category-wise Sales Analysis
3. Category-wise Order Volume Analysis
4. Size-wise Order Analysis
5. City-wise Order Analysis
 ## Power BI Dashboard
An intyerative Power BI dashboard was developed to visualize the results of the SQL analysis and provide a clear overview of Amazon sales performance. The dashboard enables users to explore key metrics and identify important business trends through visual represtentations.
### Dashboard Features
Total Orders KPI Card
Order Status Analysis
Category-wise Sales Analysis
Category-wise Order Analysis
Size-wise Order Analysis
Top Cities by Order Volume
### Benefits of the Dashboard
Provides quick insights into sales performance.
Helps identify high-performing product categories.
Displays geographical distribution of orders.
Supports data-driven business decision-making.
